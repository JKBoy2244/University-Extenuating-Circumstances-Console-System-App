#University-EC-Console-System-App

This is the project app which resembles in several ways many universities with Queen Mary University of London as one example uses. So for the app, it's basically a student to staff interaction if something happens for instance, you missed a coursework deadline or you
missed an online assessment etc for whatever reasons. The user has the chance to write down the reasons why that occurred by firstly specifying the year group he/she's in and the module/s he/she is writing for (but you can only write 1 EC at a time here). Once the user
submits the EC application, it reaches to the staff for them to see it and assess it. The staff then sends the EC outcome to the user regarding that particular module along with the report explaining that outcome.
